How to use?

There's a several ways to use edBox! Check the examples and the options below.

Example 01

example 1 (<a/>) /

Hello! This is the example 1!

html

<!--Set target element ID by attr href -->
<a class="example1" href="#target1">example 1</a>
<!--or by attr data-content in any otter element tag -->
<button class="example1" data-content="#target1">example 1</button>
<div id="target1">
	Hello! This is the example 1!
</div>

css

#target1{
	padding: 50px;
	background: crimson;
	color: #fff;
	font-size: 30px;
	display: none;
}

jQuery

//simple use
$('.example1').edbox();

Example 02

example 2

Hello again! This is the example 2!

html

<a id="example2" href="#">example 2</a>
<div id="target2">
	Hello again! This is the example 2!
</div>

css

#target2{
	padding: 20px;
	background: darkolivegreen;
	color: #fff;
	font-size: 50px;
	display: none;
}

jQuery

$('#example2').edbox({
	//set the content by option instead href attr
	content: '#target2',
	//config as needed
	overlayBg: 'darkolivegreen',
	width: 300,
	fx: 'fade',
	duration: 600,
	htmlClose: '<img src="images/close-opt-01.png"/>'
});

Example 03

example 3

html

<a id="example3" href="#">example 3</a>

css

#target3{
	padding: 30px;
	background: purple;
	color: #fff;
	font-size: 20px;
}

jQuery

$('#example3').edbox({
	//set the content as HTML string
	content: '<div id="target3">Whazzup! This is the example 3!</div>',
	duration: 1000,
	//this option require jQuery easing plugin (http://gsgd.co.uk/sandbox/jquery/easing/)
	easing: 'easeOutBounce',
	//callback a function after box close
	//this example will trigger the "example 1" after completely close
	onClose: function(){
		$.edbox({
			content: '#target1'
		})
	}
});

example 4

html

jQuery

$('#example4').edbox({
	//load an external page with parameters
	content: 'php/extpage.php?parameter=edBox',
	//config a header title if you want
	header: 'Your page title here',
	//remove the close button. In this example, we will close the box using a link like
	//<a href="#" onclick="$.edbox.close(); return false;">close</a> inside the external page.
	htmlClose: ''
});

Options

Option Default Value Description
content null DOM element, html, text, images or external pages
width null Fixed width
height null Fixed height
prefixId box Change this to create your own skin
overlayBg null Set overlay background color/image
htmlClose <img src="images/close-modal.png"/> html, text or images
header null Header content
footer null Footer content
beforeOpen function() {} Callback before open
beforeClose function() {} Callback before close
onOpen function() {} Callback after open
onClose function() {} Callback after close
disableClose false Disable / enable close function
imgLoad images/box-load.gif Load image url location
animation true Disable / enable animations
fx slide Animation type (slide, fade or toggle)
duration fast Animation duration (fast, slow or set milisseconds)
easing swing Animation easing
require jQuery easing plugin - http://gsgd.co.uk/sandbox/jquery/easing/

Methods

//Trigger by DOM element click
$('#example').edbox();

//Init box function
$.edbox();

//Close box function
$.edbox.close();
